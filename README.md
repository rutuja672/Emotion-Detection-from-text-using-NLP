# 🤖 Emotion Detection using NLP

An NLP-based Emotion Detection System that predicts emotions from text using Machine Learning techniques.

## 📌 Project Overview

This project analyzes textual input and predicts the underlying emotion using Natural Language Processing (NLP) and Machine Learning.

The application is deployed using Streamlit and provides real-time emotion prediction through an interactive user interface.

## ✨ Features

- Text Preprocessing
- TF-IDF Vectorization
- Emotion Classification
- Interactive Streamlit UI
- Real-Time Predictions
- Streamlit Cloud Deployment

## 🛠️ Technologies Used

- Python
- Natural Language Processing (NLP)
- Scikit-Learn
- TF-IDF Vectorization
- Streamlit
- Joblib

## 📊 Supported Emotions

- 😊 Joy
- 😢 Sadness
- 😠 Anger
- ❤️ Love
- 😨 Fear
- 😲 Surprise

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/yourusername/emotion-detection-nlp.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
